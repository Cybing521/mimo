# 更新日志

## 2024年11月18日 - v2.0 多进程加速版

### 🚀 重大更新：CPU多进程并行加速

#### 性能提升
- ✅ **mimo_exact_reproduction.py** 改为多进程版本
- ✅ **mimo_optimized.py** 改为多进程版本
- ✅ 加速效果：
  - 8核CPU：约 **7倍** 加速（8.3小时 → 1.2小时）
  - 16核CPU：约 **14倍** 加速（8.3小时 → 36分钟）

#### 新增功能
- ✅ 自动检测CPU核心数
- ✅ 实时进度条显示（tqdm）
- ✅ 时间戳结果保存（PNG + JSON + CSV）
- ✅ 详细的运行日志

#### 技术实现
- 使用 `multiprocessing.Pool` 并行执行1000次蒙特卡洛试验
- 每次试验完全独立，无需通信，天然适合并行
- 自动处理进程池管理和结果收集

#### 新增依赖
- `tqdm>=4.65.0` - 进度条库

#### 新增文档
- `数学原理与参数说明.md` - 详细的数学原理和参数解释
- `快速使用指南.md` - 快速上手指南
- `多进程加速说明.md` - 多进程实现细节
- `更新日志.md` - 本文件

## 2024年11月18日 - v1.1 结果保存优化

### 📁 结果管理优化

#### 主要改进
- ✅ 所有结果保存到 `results/` 目录
- ✅ 时间戳命名，避免覆盖
- ✅ 三种格式输出：PNG（图像）、JSON（数据）、CSV（表格）
- ✅ 详细的结果摘要

#### 文件命名规则
```
results/
├── exact_reproduction_YYYYMMDD_HHMMSS.png
├── exact_reproduction_YYYYMMDD_HHMMSS.json
├── exact_reproduction_YYYYMMDD_HHMMSS.csv
├── optimized_YYYYMMDD_HHMMSS.png
├── optimized_YYYYMMDD_HHMMSS.json
└── optimized_YYYYMMDD_HHMMSS.csv
```

#### 输出格式
- **PNG**: 300 DPI高分辨率图表
- **JSON**: 完整参数和结果的结构化数据
- **CSV**: Excel可直接打开的表格数据

## 升级指南

### 从v1.x升级到v2.0

#### 1. 更新代码
```bash
git pull  # 如果使用Git
```

#### 2. 安装新依赖
```bash
pip install tqdm
# 或
pip install -r requirements.txt
```

#### 3. 直接运行
```bash
python mimo_exact_reproduction.py  # 或
python mimo_optimized.py
```

**无需其他配置！** 程序会自动：
- 检测CPU核心数
- 创建多进程池
- 显示进度条
